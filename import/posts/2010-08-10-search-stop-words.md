While writing a simple search plug-in for [ExpressionEngine](https://web.archive.org/web/20100823010421/http://www.expressionengine.com/index.php?affiliate=nathanpitman) I went looking for a nicely formatted array or CSV file of ‘stop words’ which I could diff my search string against. Didn’t find anything which was an easy copy/paste so I thought I’d post the content that I [adapted from the MySQL docs](https://web.archive.org/web/20100823010421/http://dev.mysql.com/doc/refman/5.0/en/fulltext-stopwords.html) here for others to use. :)
