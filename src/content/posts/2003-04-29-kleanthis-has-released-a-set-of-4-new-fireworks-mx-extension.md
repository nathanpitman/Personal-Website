Kleanthis has released a set of 4 new Fireworks MX extensions. I downloaded these some time ago, but totally forgot to report on them, they are:

- [Crop Assistant](https://web.archive.org/web/20100113001142/http://www.projectfireworks.com/Lab.aspx?show=extension_crop_assistant.html)
- [Fill Handles Assistant](https://web.archive.org/web/20100113001142/http://www.projectfireworks.com/Lab.aspx?show=extension_fill_handles_assistant.html)
- [Knife Assistant](https://web.archive.org/web/20100113001142/http://www.projectfireworks.com/Lab.aspx?show=extension_knife_assistant.html)
- [Rectangle Assistant](https://web.archive.org/web/20100113001142/http://www.projectfireworks.com/Lab.aspx?show=extension_rectangle_assistant.html)

As you’d expect from Kleanthis, these extensions are brilliantly put together, work well and look great. Worth the download.
