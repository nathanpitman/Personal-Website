A while back the lads over at salted gave us a [sneak peek](http://www.salted.com/unsalted/cssvista-a-sneak-preview) at CSSVista, their standalone Windows application for web developers which allows you to edit raw CSS whilst live previewing the results in a tabbed IE and Firefox browser interface.

Well [CSSVista is now available](http://www.sitevista.com/cssvista/), even better, it’s **free**. Go grab yourself a copy and give it a whirl. I think this liitle application has some great potential! :)
