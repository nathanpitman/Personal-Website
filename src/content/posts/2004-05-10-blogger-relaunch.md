Following the acquisition of [Blogger](https://web.archive.org/web/20100107204601/http://www.blogger.com/) in 2003 by [Google](https://web.archive.org/web/20100107204601/http://www.google.com/), Blogger has relaunched with a [rich new feature set](https://web.archive.org/web/20100107204601/http://www.blogger.com/knowledge/2004/05/great-blogger-relaunch.pyra) and a very chic new CSS driven look.

![The chic new CSS driven Blogger site](/images/13.jpg "The chic new CSS driven Blogger site")

If you’re looking to set up a blog then I’d say you couldn’t go far wrong with Blogger, although of course if you have access to MySQL on a Unix box I’d have to suggest [Textpattern](https://web.archive.org/web/20100107204601/http://www.textpattern.com/). ;)
