[Kleanthis Economou](https://web.archive.org/web/20100106033728/http://www.projectfireworks.com/) has posted an updated [Fireworks Commands Wrapper](https://web.archive.org/web/20100106033728/http://www.macromedia.com/cfusion/exchange/index.cfm#loc=en_us&view=sn111&viewName=Flash%20Extension&authorid=8645700&page=0&scrollPos=0&subcatid=0&snid=sn111&itemnumber=13&extid=1013904&catid=0) for Flash on the Macromedia Exchange. This allows you to author Fireworks Commands or Command Panels in Flash by addressing the Fireworks API using Acionscript.
